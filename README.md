# 🏥 National Health Data Analysis Dashboard

## 📌 Overview

This project presents an interactive **Healthcare Data Analysis Dashboard** that provides insights into patient demographics, hospital performance, and treatment costs.

The dashboard helps analyze key healthcare metrics such as admissions, billing, patient age, and length of stay.

---

## 📊 Dashboard Features

### 🔹 Key Trends

* Total admitted patients and growth comparison
* Rooms/bed capacity analysis
* Average billing amount
* Patient age insights
* Admission trends by month and type
* Length of stay (LOS) distribution

---

### 🔹 Patient Demographics

* Patient distribution by age group and gender
* Medical condition analysis (e.g., Cancer)
* Hospital-wise patient comparison
* Blood group and demographic breakdown

---

### 🔹 Treatment & Cost Analysis

* Total billing by hospital and condition
* Cost comparison by admission type (Emergency, Urgent, Elective)
* Medication usage across conditions
* LOS-based billing trends

---

## 🛠️ Tools & Technologies

* Power BI
* Data Visualization
* Data Cleaning & Transformation
* Excel / CSV Dataset

---

## 📈 Key Insights

* Significant variation in billing across hospitals
* Extended stays contribute most to total costs
* Certain conditions (e.g., Diabetes, Hypertension) dominate patient volume
* Seasonal/monthly trends visible in admissions
